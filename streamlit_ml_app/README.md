# 🎥 Live Object Detection & Tracking

Real-time object detection web app using **Streamlit + YOLOv8 + Webcam**

[![Streamlit App](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://share.streamlit.io/)

## ✨ Features

✅ **Real-time object detection** with YOLOv8  
✅ **Smooth object tracking** across frames  
✅ **Object counting** (people, phones, etc.)  
✅ **Security alerts** for specific objects  
✅ **Frame saving** functionality  
✅ **Live statistics dashboard**  
✅ **Confidence threshold control**  
✅ **Professional UI** with real-time metrics
